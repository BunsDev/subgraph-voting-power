# GNO Voting Power
This repo is a collection of subgraphs which aggregate the balance of GNO held in multiple multiple contracts on several networks.

Currently, it tracks GNO balances in:
* Ethereum Mainnet:
  - [x] GNO
  - [x] lGNO
  - [ ] Uniswap v3
  - [ ] Balancer
* Gnosis Chain:
  - [x] GNO
  - [x] mgno
  - [x] lgno
  - [ ] staking deposits
  - [ ] 1Hive
  - [ ] swapper
  - [ ] sushiswap
  - [ ] elk
  - [ ] baoswap
  - [ ] symetric
  - [ ] agave
